Proyecto Metodología es una plataforma web diseñada para proporcionar información básica sobre psicología dirigida a estudiantes. La herramienta fue desarrollada utilizando HTML, CSS, y JavaScript, ofreciendo funciones interactivas y atractivas que permiten a los usuarios explorar diversos temas de manera dinámica. Esta plataforma educativa se destaca por su facilidad de uso y su enfoque en hacer el aprendizaje accesible y llamativo para los estudiantes.
